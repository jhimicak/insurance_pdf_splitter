# PDF Splitter & Masker + 이메일 자동 발송 안내

이 프로그램은 다중 페이지 PDF를 개별 페이지로 분리 및 마스킹하고, 엑셀 명단을 기반으로 각 직원에게 이메일을 자동 발송하는 도구입니다.

## 1. 사전 준비 (필수 라이브러리 설치)

프로그램 실행을 위해 아래 라이브러리 설치가 필요합니다. 터미널(CMD 또는 PowerShell)에서 다음 명령어를 입력하세요.

```bash
pip install -r requirements.txt
```

## 2. 실행 방법

`app.py` 파일을 실행하면 GUI 창이 나타납니다.

```bash
python app.py
```

## 3. 주요 기능

### 탭 1: PDF 분할 & 마스킹
- **성명**: '홍길동' -> '홍*동' (가운데 글자 마스킹)
- **증번호**: 2자리당 1개 '*' 처리
- **주민등록번호**: 뒷자리 첫 번째 숫자 외 모두 '*' 처리
- **파일 저장**: `[연번]-[성명].pdf` 형식

### 탭 2: 이메일 발송
- **엑셀 매핑**: `연번`, `성명`, `이메일` 컬럼이 포함된 엑셀 파일을 사용하여 대상자 매칭
- **SMTP 설정**: Gmail(앱 비밀번호 필요) 또는 사내 메일 서버 지원
- **자동 발송**: 생성된 PDF 파일명을 분석하여 엑셀의 이메일 주소로 자동 첨부 및 발송

## 4. 엑셀 파일 형식 예시

| 연번 | 성명 | 이메일 |
| :--- | :--- | :--- |
| 65 | 임재혁 | jhim@example.com |
| 66 | 임연혁 | yh@example.com |

## 5. Gmail용 '앱 비밀번호' 설정 안내
구글 계정의 2차 인증을 사용하는 경우, 일반 비밀번호로는 로그인이 거부됩니다.
1. [구글 계정 설정] -> [보안] -> [2단계 인증] 활성화
2. 가장 하단의 **'앱 비밀번호'** 메뉴 선택
3. 앱 이름(예: "PDF Splitter")을 입력하고 생성
4. 생성된 **16자리 비밀번호**를 프로그램의 비밀번호 칸에 입력하세요.
